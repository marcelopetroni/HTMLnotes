OBS IMPORTANTE: antes vamos entender o que é HTTP? sigla para Hyper Text Transfer Protocol. - é o protocolo que que define regra de troca de informações entre servidores que abrigam sites e os computadores. permite transferir hiper-texto de um lado a outro, onde um navegador (cliente) estabelece a comunicação com um servidor da World Wide Web (WWW). A mensagem enviada pelo cliente é chamada de solicitação (request), Primeiramente, o navegador descobre o endereço IP do servidor web usando o DNS. Em seguida, o navegador web faz a conexão diretamente com o servidor web e é enviado uma mensagem chamada de resposta (response), o conteúdo solocitado. Todas as mensagens enviadas pelo protocolo (HTPP) através de uma interface (site ou página que geralmente utiliza HTML) são localizadas na rede através das URLs (Localizador Uniforme de Recursos), processadas e retornadas para os usuários. - foi criado para garantir que a comunicação entre o PC e os servidores será criptografada. - Um formulário é definido pela tag form, e seus atributos action e method:
