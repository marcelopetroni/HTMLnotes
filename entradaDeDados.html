input type = "password" - Esconde o que está sendo escrito no campo. - Atributos: minlength -> nº mínimo de caracteres. maxlength -> nº máximo de caracteres. pattern -> validação do que está sendo digitado. ex: pattern="[A-Za-z]{3}-[0-9]{4}", uma placa e carro no formato ABC4578 (três letras e 4 números) - use title para exibir mensagem que com as exigências da senha. placeholder -> texto clarinho dentro do campo apenas ilustrativo. disabled -> bloqueia campo. required -> campo obrigatório para preenchimento do campo. inputmode = "numeric" para abrir teclado de números. autocomplete = "on" -> sugere se tiver uma senha salva para ser autopreenchido. autocomplete = "new-password" -> dependendo do navegador, sugere uma senha para autopreencher.
