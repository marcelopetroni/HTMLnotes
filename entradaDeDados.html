input type = "url" - irá validar se o valor for uma url. - Atributos: placeholder, disabled/readonly, required, minlength/maxlength, size, list value -> texto pronto como sugestão já escrito no campo. pattern -> validação. ex: pattern = ".*\.com.*" onde ".*" o ponto (.) representa qualquer caractere, exceto uma nova linha e o asterisco (*) indica que o caractere anterior pode aparecer zero ou mais vezes. "\." para aceitar o ponto literalmente e não como um significado diferente. "com" o que normalmente tem nas urls podendo ter ou não "\.br" sem seguida. ".*" no final caso tenha algum diretorio extra como "/discover/curso". spellcheck -> verificação ortográfica para aquele input. input type = "file" - irá pedir para seelecionar um arquivo. - usa a notação "enctype="multipart/form-data" dentro da tag form para poder enviar arquivo. - Atributos: value -> representa o path do arquivo selecionado. accept-> para apenas aceitar e ilustrar arquivos de certo tipo, exemplo: accept = "image/*" accept = "image/png" accept = ".png" accept = "video/*" accept = ".html" multiple -> para poder selcionar mais de um arquivo.
